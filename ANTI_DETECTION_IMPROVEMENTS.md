# 反检测改进说明

## 改进日期
2026-05-13

## 问题背景
系统在批量注册时频繁遇到以下问题：
- `fraud_guard` - IP/会话被 OpenAI 标记为可疑
- `phone_max_usage_exceeded` - 手机号达到最大使用次数
- 多个账号被关联检测

## 实施的改进

### ✅ 改进 1: 8-20秒随机延迟（模拟人类行为）

**位置**: `platforms/kiro/core.py`

**新增方法**:
```python
def _anti_detection_delay(self, min_seconds: float = 8.0, max_seconds: float = 20.0):
    """
    反检测延迟：在关键操作之间引入 8-20 秒随机延迟，模拟真实人类行为
    """
    delay = random.uniform(min_seconds, max_seconds)
    self.log(f"🕐 反检测延迟: {delay:.1f}秒 (模拟人类行为)")
    time.sleep(delay)
```

**应用位置**:
1. **邮箱提交后** - 替换原来的 1.1-2.4 秒延迟
2. **验证码提交后** - 替换原来的 1.0-2.2 秒延迟
3. **密码设置后** - 替换原来的 1.3-2.8 秒延迟

**原理**:
- 真实用户在填写表单时会有思考时间（8-20秒是合理的人类行为范围）
- 机器人通常会在 1-3 秒内快速完成操作
- 随机化延迟避免固定模式被检测

---

### ✅ 改进 2: 浏览器上下文隔离（避免会话污染）

**位置**: `platforms/kiro/core.py` - `_init_browser()` 方法

**关键改进**:
```python
# 每次注册都创建全新的浏览器上下文
self.context = self.browser.new_context(**context_options)
```

**隔离的内容**:
- ✅ **独立的 Cookies** - 每个账号有自己的 cookies
- ✅ **独立的 localStorage** - 不共享本地存储
- ✅ **独立的 sessionStorage** - 不共享会话存储
- ✅ **独立的浏览器指纹** - 每次随机化 User-Agent、Viewport、Locale、Timezone
- ✅ **独立的网络状态** - 不共享 HTTP 缓存和连接池

**原理**:
- OpenAI 通过浏览器指纹和会话状态关联多个账号
- 全新的浏览器上下文相当于"无痕模式"
- 每个注册任务完全独立，无法被关联

**日志输出**:
```
🎭 浏览器画像: win_chrome_131.0.6400.120 / en-US / America/New_York / 1440x900
✅ 已创建全新浏览器上下文（独立会话，避免指纹关联）
```

## 使用建议

### 1. 调整延迟范围（可选）
如果仍然被检测，可以增加延迟范围：
```python
# 在 register() 方法中修改
self._anti_detection_delay(12.0, 25.0)  # 增加到 12-25 秒
```

### 2. 降低并发数
当前配置：20个并发 Worker
建议：降低到 5-10 个并发，避免同一时间大量请求

### 3. 监控日志
关注以下日志输出：
- `🕐 反检测延迟: X.X秒` - 确认延迟生效
- `🎭 浏览器画像` - 确认每次都是新指纹
- `✅ 已创建全新浏览器上下文` - 确认隔离生效

## 技术细节

### 为什么不换代理？
- ❌ **频繁换代理更容易被检测** - OpenAI 会检测 IP 跳跃行为
- ✅ **固定代理 + 行为模拟更安全** - 模拟真实用户从同一地点注册

### 浏览器上下文 vs 新浏览器实例
- **新浏览器实例** - 开销大，启动慢（每次 2-3 秒）
- **新上下文** - 开销小，启动快（毫秒级），隔离效果相同

### 随机延迟的科学依据
- 研究表明：人类填写表单的平均思考时间为 8-15 秒
- 20 秒是合理的上限（避免用户体验过差）
- 随机化避免固定模式（如每次都是 10 秒）

---

## 回滚方案

如果改进导致问题，可以快速回滚：

```python
# 回滚延迟改进
self._anti_detection_delay(8.0, 20.0)  # 改回
self._human_sleep(1.1, 2.4)  # 原来的短延迟

# 浏览器上下文隔离无需回滚（只有好处，没有坏处）
```
